# 《开放时代》投稿体例

# (2026年1月1日起施行)

# 一、题注、图表等

# (一) 题注

1. 如系基金、课题等资助项目，可以标注项目名称和编号。  
2. 致谢语尽量简短。

# （二）内容提要、关键词

内容提要不超过350字，关键词3—5个。

# （三）语言文字

正文中的外国人名、地名等专有名词应当翻译，需要使用外国语言文字的，应当用中文解释。

# （四）小标题

各级小标题的序号依次为：一 /（一）/1 /（1）。

# （五）图形、表格、照片

1. 图表的标题用阿拉伯数字，全文连续编号，置于图表上方。   
2. 表格要显示所有框线，图表均为黑白效果。  
3. 照片的说明应简明扼要，并标注拍摄者或来源。

# 四、夹注、参考文献

# (一) 规则

1. 夹注用括号标明，可以加上首次出版年份、页码。

( Wrigley & Schofield, 1989[1981]: 208-209 )

2. 只列举夹注对应的参考文献，置于尾注之前。

3. 参考文献与尾注的写法近似，但不编序号。西文文献将责任者（第一责任者）的姓氏提前。

陈少明，2005，《经典世界中的人、事、物——对中国哲学书写方式的一种思考》，载《中国社会科学》第5期。

《第四期中国妇女社会地位调查主要数据情况》，2021，载《中国妇女报》12月27日，第4版。MacFarquhar, Roderick & Michael Schoenhals, 2006, Mao's Last Revolution, Cambridge, Mass.; London: Belknap Press of Harvard University Press.

# （二）顺序

1. 西文文献按责任者姓氏字母排序，中文和日文文献按姓氏的汉语拼音排序。

2. 不同语种的文献，按以下顺序排列：中文 / 日文 / 英文 / 其他西文。

# 五、数字、标点符号

# (一) 中文

# 1. 国家标准（下载链接）

(1)《标点符号用法》（GB/T 15834-2011）。  
(2)《出版物上数字用法》（GB/T 15835-2011）。

# 2. 时间

（1）公历的世纪、年代、年、月、日、时、分、秒，使用阿拉伯数字。  
（2）历史朝代纪年、传统上采用汉字形式的非公历纪年，以阿拉伯数字括注公历。  
太平天国庚申十年九月二十四日（清咸丰十年九月二十日，1860年11月2日）  
藏历阳木龙年八月二十六日（1964年10月1日）

# 3. 数字

（1）表示数值的范围时，使用一字线“一”。小数点前的“0”不能省略。  
0.10千克—0.15千克  
(2) 必须采用汉字数字的情形: 已定型的词语、成语、惯用语、缩略语; 需要突出庄重典雅的表达效果; 数字连用的概数、含 “几” 的概数; 干支纪年、农历月日、历史朝代纪年等。

# （二）英文

1. 英文标点一律使用半角符号。  
2. 省略号为三个点，位置居下，前后不用空格。  
He seemed nervous...stared straight ahead...kept twitching and jerking...then he ran for shelter.   
She disagreed with the decision....

3. 破折号相当于两个连字符（-），前后均空格。

Then the proposals – both Xiao Yang's and mine – were adopted.
